Carrot2 4.8.x

Release history for Carrot2 4.8.0 and bugfix releases.

Version 4.8.2

The 4.8.2 release fixes the user interface to correctly pass the user-provided customer ID to the eTools search service.

Version 4.8.0

This release includes an upgrade of dependencies, build plugins and a switch to Java 21 as the minimum runtime platform (a consequence of switching to Lucene 10).

Dependency upgrades

jackson to 2.19.0, jetty to 11.0.25, Lucene to 10.2.1, slf4j to 2.0.17, log4j to 2.24.3 (and other transitive updates).
